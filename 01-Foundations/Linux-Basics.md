Linux Fundamentals:

To work and survive in a security environment you must be able to move around the system and look at files using the command line interface (CLI).

Navigation Commands:
pwd (Print Working Directory): Tells Us the location where we are.
ls (List): Shows the files and folders in your current directory.
ls -la: flags of ls shows hidden files and detailed file permissions.
cd <folder_name> (Change Directory): Moves from one folder to another.
cd .. : Moves one folder back.

File Operations:
cat <file_name> (Concatenate): Displays the contents of a file.
mkdir <directory_name> (Make Directory): Creates a new folder.
cp <source> <destination> (Copy): Copies a file from one place to another.
mv <source> <destination> (Move): Moves or renames a file.

The Security Power Tool
grep (Global Regular Expression Print): Searches for specific text inside a file. In security we use this constantly to find malicious activity hidden inside thousands of lines of system logs.
